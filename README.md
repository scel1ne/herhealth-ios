# HerHealth iOS

A women's health companion app for iOS, built with Swift and SwiftUI.

## Tech Stack

- Language: Swift
- UI Framework: SwiftUI
- Platform: iOS
- Health: HealthKit

## Project Structure

```
HerHealth/
├── HerHealth/                  # App source code
│   ├── App/                    # App entry, Info.plist
│   ├── Home/                   # Home module
│   ├── Chat/                   # AI chat
│   ├── Journal/                # Journal
│   ├── BodyCheck/              # Body check
│   ├── Stats/                  # Statistics
│   ├── Learn/                  # Learn center
│   ├── Quiz/                   # Quizzes
│   ├── Library/                # Saved content
│   ├── Calm/                   # Breathing & meditation
│   ├── Me/                     # Profile
│   ├── Settings/               # Settings
│   ├── Models/                 # Data models
│   ├── Components/             # Shared UI components
│   ├── Theme/                  # Theme & styles
│   └── Resources/              # Assets (icons, colors)
├── HerHealth.xcodeproj/        # Xcode project
└── generate_pbxproj.py         # Project file generator
```

## Getting Started

```bash
open HerHealth.xcodeproj
```

Requirements:
- Xcode 15+
- iOS 16+
